### Pitang Images

## <a href="https://pitang-images.netlify.app">DEMO</a>

## Quick start

Install:

```
git clone https://github.com/gabriel-cardoso-oliveira/pitang-images.git
```
```
cd pitang-images/
```
```
yarn install
```

Execute:

```
yarn start
```

Test:

```
yarn test
```
